CS 460 Homework 1: HTML, CSS and Bootstrap

This collection of webpages is in complience with our CS 460 Fall Term 2017 class at Western Oregon University. Its main focus is to allow us, the students, an opportunity to further develop our knowledge and skills in HTML, CSS and Bootstrap.




Assignment Requirements

  • Must use Bootstrap for the layout of all pages and must demonstrate both single column layout and two or more column formatting
  • Must have a separate CSS file (e.g. styles.css) in which some of the custom classes are written and must be used
  • A navigation bar or menu that contains links to all the pages is needed
  • At least one table is needed
  • One of each kind of list is needed: ol, ul and dl
  • All main elements need to be styled consistently: it is allowed to use those defined by Bootstrap or customized ones. No obnoxious colors or backgrounds allowed. Attempt to make a professional and modern looking page
  • All code must be written by the student and may not use a WYSIWYG designer such as Dreamweaver. Good (free) HTML and CSS aware editors include Atom, Brackets, Visual Studio Code, Notepad++ and Textmate (OS X only)
